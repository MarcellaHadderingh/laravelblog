# laravelblog.
Mijn aller eerste laravel opdracht tijdens de bootcamp.

Hij is nog niet af.

### Het bevat het volgende:
- Informatie over de bootcamp.
- Blog lezen en schrijven.
- Registreren en inloggen.
- Avatar wijzigen.

### Hij is opgebouwd in:
- Laravel
- HTML/CSS
- Bootstrap
- Zelf Design van logo en Foto's
- PHP
- MYSQL
